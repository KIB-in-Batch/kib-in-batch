Introduction
============

KIB in Batch behaves similarly to a Linux system, but with some differences and additions.
Here you can see how to remove KIB in Batch from your system.

Uninstallation
==============

You should first download the source code archive from <https://github.com/KIB-in-Batch/kib-in-batch/releases/latest>, extract it and run this in PowerShell or Command Prompt in the directory you extracted the archive to:

```code
.\uninstall.bat
```endcode

End
===

After following this, you should have succesfully removed KIB in Batch from your system.
